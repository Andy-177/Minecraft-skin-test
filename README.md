# Minecraft-skin-test
用于测试的皮肤
# 使用方法
下面方位描述以人物的正视图为准

如果你需要同时检查任务部位和方位的问题，建议使用test_all
## 部位
如果只是想使用测试皮肤测试人物部位的错位建议使用test_lite，下面是部位与颜色的对应关系
|部位|颜色|
|----|----|
|头部|红色|
|身体|青色|
|左手|绿色|
|右手|蓝色|
|左腿|黄色|
|右腿|紫色|
## 方位
如果还要测试皮肤方位是否正确建议使用test_face，下面是方位与颜色的对应关系
|方位|颜色|
|----|----|
|正面|红色|
|背面|青色|
|左边|绿色|
|右边|蓝色|
|上面|黄色|
|下面|紫色|
